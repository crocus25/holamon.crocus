# holamon
## En quant a aquest repositori
Aquest repositori no té gairebé cap utilitat més que fer proves i aprenentatges de git.
Això no treu que els arxius i codi contingut en aquest repositori puguin ser descarregats i fets servir per qualsevol altre projecte propi o alíé.
Sense ànims de molestar ningú, aquest projecte fa servir el català com a llengua vehicular.
